# Fortanix KMIP server configuration

`pg_tde` is compatible with Fortanix Data Security Manager (DSM) via the KMIP protocol. For a full setup guide, see [the Fortanix KMIP documentation here](https://support.fortanix.com/docs).

For more information on adding or modifying the provider, see the [Adding or modifying KMIP providers](https://docs.percona.com/pg-tde/functions.html?pykmiph=pg_tde_add_global_key_provider_kmip#add-or-modify-kmip-providers) topic.
